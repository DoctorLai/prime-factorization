# Security Policy

## Supported Versions

This project is a small static React application. Security fixes are applied to the `main` branch and released through the GitHub Pages deployment workflow.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Report security concerns privately through GitHub Security Advisories for this repository, or contact the maintainer through the profile links on [github.com/doctorlai](https://github.com/doctorlai).

Please include:

- A short description of the issue.
- Steps to reproduce or a proof of concept.
- The affected browser, Node.js version, or dependency version when relevant.
- Any suggested mitigation, if known.

The maintainer will review reports as availability allows and prioritize issues that affect users of the deployed app or the project supply chain.
