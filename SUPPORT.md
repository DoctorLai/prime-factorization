# Support

## Getting Help

- For bugs, open a GitHub issue with reproduction steps, browser details, and the expected result.
- For feature ideas, open a feature request and describe the use case.
- For security concerns, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

## Project Scope

Prime Factorization is a small browser-based calculator for positive integers and a reference list of primes up to 2000. Questions and issues are easiest to handle when they relate to:

- Factorization correctness.
- Browser behavior and accessibility.
- Build, test, deployment, and documentation workflows.
- Dependency maintenance.
