# Privacy

Prime Factorization is a static client-side app.

## Data Collection

The app does not send entered numbers to an application server. Factorization runs in the browser.

## Local Storage

The app stores the dark mode preference in `localStorage` under the `darkMode` key and the last entered number under the `primeFactorizationNumber` key. These values stay in the user's browser and can be cleared through browser site-data settings.

## Third Parties

When using the deployed GitHub Pages site, GitHub may process standard web request data according to GitHub's own policies. External links in the README or footer navigate away from this project.

## Contact

Open a GitHub issue for privacy-related documentation corrections. Report security-sensitive privacy issues through [SECURITY.md](SECURITY.md).
